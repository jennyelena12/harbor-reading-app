================================================================================
                          QUICK START GUIDE
================================================================================

⏱️  TAKES 5 MINUTES

Step 1: Create Xcode Project (1 minute)
─────────────────────────────────────────
1. Open Xcode
2. File → New → Project
3. Choose "App" template
4. Name: Harbor
5. Interface: SwiftUI
6. Click Create

Step 2: Add Swift Files (1 minute)
──────────────────────────────────────
1. Drag these 7 files into Xcode project:
   • HarborApp.swift
   • Models.swift
   • Managers.swift
   • HomeScreen.swift
   • ReadingScreen.swift
   • FavoritesGalleryView.swift
   • HapticManager.swift
2. Check "Copy items if needed"

Step 3: Configure Permissions (1 minute)
─────────────────────────────────────────
1. Open Info.plist
2. Add these 3 keys:
   • NSCameraUsageDescription: 
     "Harbor uses your camera's light sensor..."
   • NSMicrophoneUsageDescription: 
     "Harbor uses your microphone to recognize..."
   • NSSpeechRecognitionUsageDescription: 
     "Harbor requests permission to use speech..."

Step 4: Enable Capabilities (1 minute)
──────────────────────────────────────
1. Select "Harbor" target
2. Signing & Capabilities tab
3. "+ Capability" button
4. Add: Microphone
5. Add: Camera

Step 5: Build & Run (1 minute)
──────────────────────────────
1. Press: Cmd + R
2. Select: iPhone 17 simulator
3. Watch it launch!

🎉 DONE! Harbor is running.

================================================================================
                       KEY DESIGN DECISIONS
================================================================================

1. NO EXTERNAL DEPENDENCIES
   ✓ Maximum control, faster updates, simpler deployment
   ✓ Uses only native iOS frameworks

2. USERDEFAULTS FOR STORAGE
   ✓ Simple, fast, perfect for small data
   ✓ Easy to swap for Core Data later

3. CAMERA-BASED LIGHT DETECTION
   ✓ Works on all iPhones, no special sensor needed
   ✓ Processes on background thread

4. MVVM ARCHITECTURE
   ✓ Clean separation of concerns
   ✓ Easy to test and extend

5. OCEAN THEME THROUGHOUT
   ✓ Visual metaphor of literary journey
   ✓ Accessible, beautiful 4-color palette

6. HAPTIC + ANIMATION FOCUSED
   ✓ Feels like a native Apple app
   ✓ 120fps smooth animations

================================================================================
                      TROUBLESHOOTING QUICK FIXES
================================================================================

"App crashes on launch"
→ Verify all 7 .swift files are in project

"Permissions not requesting"  
→ Settings → Harbor → Enable permissions OR
→ Reset simulator: Erase All Content and Settings

"Microphone not working"
→ Settings → Harbor → Microphone → ON
→ Need internet for best voice recognition

"Brightness not adjusting"
→ Settings → Harbor → Camera → ON
→ Try different light conditions (bright and dark)

"Can't hear voice feedback"
→ Settings → Harbor → Microphone → ON
→ Check volume isn't muted
→ Try with different audio

More Issues?
→ See SETUP.md Troubleshooting section
→ Check TESTING_GUIDE.md for detailed procedures
