TROUBLESHOOTING
================================================================================

"XcodeGen not found"
  → Install Homebrew: /bin/bash -c "$(curl -fsSL ...)"
  → Then: brew install xcodegen

"Cannot find 'Book' in scope"
  → Click file in Xcode
  → File Inspector (right) → Check "Harbor" target membership

"Microphone not working"
  → Settings → Privacy → Microphone → Harbor: Allow

"Voice not recognized"
  → Speak clearly, simulator has limitations
  → Much better on real device
  → Try: "next page", "chapter 2", "previous"

"Brightness not adapting"
  → Settings → Privacy → Camera → Harbor: Allow
  → Better on real device with actual light sensor
  → Try in dark room to see effect

More help:
  → See "Troubleshooting" in 00_START_HERE.md
